🏗 Architecture
MAIN calls fbLocalSystemTime once, then passes systemTime to each logger.

3 × FB_SqlLogger — each holds its own FB_SQLDatabaseEvt + FB_SQLCommandEvt. All three connect independently on startup and stay connected.

SELECT state machine — lives directly in MAIN. Uses a separate FB_SQLDatabaseEvt that opens and closes per query, so it does not compete with the INSERT connections.

GVL_DB centralises shared constants: DB_ID=1, C_MaxSelectRows=1000, arrResults[], nResultCount, bWriteError.
🗄 MySQL Table Schema
CREATE TABLE ds ( id INT NOT NULL, timestamp DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP, value FLOAT NOT NULL ); -- Example INSERT generated at runtime: INSERT INTO ds(id,value) VALUES (1, 22.5); -- Example SELECT: SELECT id, timestamp, value FROM ds ORDER BY timestamp DESC LIMIT 1000;
